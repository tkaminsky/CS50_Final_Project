Instructions

Welcome to matrix mate! Here's a brief summary of the capabilities of each component of this website:

Matrix Transformation Visualizer

The Matrix Transformation Visualizer allows you to input any 2 by 2 matrix with real entries and visualize the result. Any invalid responses, including non-real entries, will be interpreted as the value of the placeholder in each box, either 0 or 1. The output will show both the untransformed and the transformed gridlines. For the untransformed graph, see the key under the gridlines to see what the resolution is. All gridlines in the transformed graph correspond to unit gridlines.

Vector Visualizer

The Vector Visualizer allows you to input up to five two-dimensional vectors and see them displayed on a result screen. All vectors are drawn beginning at the origin. Consult the key to see what the resolution of the output graph is.

Vector Operations Visualizer

The Vector operations visualizer allows you to see some of the fundamental operations on vectors. As always, the resolution can be determined by observing the key below the graphs. Supported operations are:

  • Vector Addition: Input two vectors and see their sum displayed in blue on a result screen, alond with the original vectors in red.
  • Scalar Multiplication: Input a vector and a scalar and see the two vectors displayed on the result screen, with the result vector in blue.
  • Orthogonal Decomposition: Input a vector and see its orthogonal decomposition with respect to the standard basis displayed on the result screen, with both vectors in red.
